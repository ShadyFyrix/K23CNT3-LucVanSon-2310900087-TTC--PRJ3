Luồng Xác thực (Authentication Flow)

ADMIN Flow

1. Login tại /LvsAdmin/LvsLogin
2. Kiểm tra role = ADMIN
3. Redirect → /LvsAdmin/LvsDashboard

MODERATOR Flow

1. Login tại /LvsUser/LvsLogin
2. Kiểm tra role = MODERATOR
3. Redirect → /LvsModerator/LvsDashboard

USER Flow

1. Login tại /LvsUser/LvsLogin
2. Kiểm tra role = USER
3. Redirect → /lvsforum

Tính năng Bảo mật

Role-based access control (RBAC)
Session management với HttpSession
Password validation
Account status checking (BANNED/INACTIVE)
